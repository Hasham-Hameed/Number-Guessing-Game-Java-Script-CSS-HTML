🎯 Guess the Number Game

A simple and interactive Guess the Number game built using HTML, CSS, and JavaScript.

📌 Description

This is a browser-based game where:

The computer generates a random number between 1 and 100
The player tries to guess the number
The game gives hints like:
📉 Too high
📈 Too low
The game ends when the correct number is guessed
🚀 Features
🎲 Random number generation
📥 User input via input box (no prompt)
💬 Instant feedback messages
🎨 Styled UI using CSS
🔢 Attempt counter
🛠️ Technologies Used
HTML
CSS
JavaScript
▶️ How to Run

Copy the code into a file named:

index.html
Open the file in your browser:
Double click the file
OR
Right click → Open with browser
🎮 How to Play
Enter a number between 1 and 100
Click Submit
Follow the hints:
Too high → guess lower
Too low → guess higher
Keep trying until you guess correctly 🎉
